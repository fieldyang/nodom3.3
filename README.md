#### 从命令行创建一个新的仓库

```bash
touch README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://git.trustie.net/leslieder/nodom.git
git push -u origin master

```

#### 从命令行推送已经创建的仓库

```bash
git remote add origin https://git.trustie.net/leslieder/nodom.git
git push -u origin master

```
